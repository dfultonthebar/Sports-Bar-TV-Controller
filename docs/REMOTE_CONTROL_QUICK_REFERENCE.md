## Troubleshooting Quick Tips

### Remote Not Working
1. Check device is "Online" in system
2. Try command again (may have timed out)
3. Check IR emitter position (for IR devices)
4. Use physical remote as backup

### Wrong Device Responds
1. Verify correct device selected in system
2. Check IR emitters not interfering
3. For CEC, check device address mapping

### Commands Delayed
1. Network lag possible
2. Check WiFi signal on phone/tablet
3. Check device response time
4. Contact admin if persistent

## Tips & Best Practices

### Efficient Control
- **Use Channel Presets:** Faster than manual entry
- **Use Sports Guide:** Find games quickly
- **Learn Common Channels:** Memorize ESPN, FS1, etc.

### Backup Plan
- **Physical remotes available:** Always have backup
- **Know device locations:** Which cable box = which TV
- **Document issues:** Note problems for admin

### Customer Service
- **Quick responses:** Change channels promptly
- **Know limitations:** Not all games available
- **Offer alternatives:** Different TV or streaming
