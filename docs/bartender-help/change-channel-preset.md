# Changing a Channel or Loading a Preset

If a TV is on the wrong channel or you need to load a saved preset:

1. On the bartender iPad, open the TV control screen.
2. Find the TV that's wrong.
3. Tap "Change Channel" or the preset buttons (like "ESPN", "FOX", "Game 1").
4. Confirm the change – the DirecTV box will switch.

The system uses IR to the DirecTV receivers for channel changes. Presets are saved in the app for common sports channels.

If the remote isn't responding, see remote-isnt-working.md instead.

Escalate to manager if the box is offline.
