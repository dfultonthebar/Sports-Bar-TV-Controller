# Adding or Replacing a DirecTV Box

**When to use:** A DirecTV receiver died, was swapped by the tech, or you need to hook up a new one for extra TVs.

**What it looks like:** Black or silver box (usually on the rack or shelf near the TVs). Has power light, Ethernet cable plugged in back, HDMI out to the matrix switcher.

**Bartender steps:**

1. Tell the manager or text the tech lead — they handle the actual box swap and network setup. You do NOT plug cables yourself.

2. Once the box is in and powered on (blue light), the system should auto-detect it within a few minutes.

3. Check the TV that uses that box: go to the iPad remote, pick "TVs" tab, find the affected output, and re-route it to the new DirecTV input if needed (see WRONG_CHANNEL_ON_TV.md for routing steps).

4. Test by tuning a channel on that TV using the preset buttons.

**If it doesn't show up:**
- Confirm the Ethernet light is on (solid green).
- Power cycle the box: unplug power for 30 seconds, plug back in.
- If still missing after 10 minutes, text the manager with the box location and "DirecTV not detected."

You can't break anything by checking lights or re-routing on the iPad. Escalation: manager or on-call tech.
