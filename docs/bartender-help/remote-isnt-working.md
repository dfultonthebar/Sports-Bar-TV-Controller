# Remote Isn't Working

If the remote for a TV or DirecTV box stops responding:

1. Point the remote directly at the silver box on the top rack (IR blaster location).
2. Make sure batteries are good - swap if the light is dim.
3. For TVs: use the power button on the side or back of the TV to turn it off/on.
4. Tell the manager if it keeps happening - they can check the IR blaster.

You can't break anything by trying the power button.
