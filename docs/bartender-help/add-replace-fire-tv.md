# Adding or Replacing a Fire TV Streaming Device

This is for when a Fire TV box dies or you need to add a new streaming device to a TV.

**What it looks like:** The TV shows nothing, a black screen, or the wrong app, and it's not a cable box issue.

**You can't break anything by trying these steps.**

1. Find the black Amazon Fire TV Cube (small square box, usually on a shelf near the TV or in the rack). Note which TV number it belongs to (label on the box or cable).

2. Power it off: unplug the power cord from the wall or the box for 30 seconds, then plug back in. Wait for the light to come on and the home screen to load.

3. If it's a full replacement: Unplug the old box completely. Plug the new Fire TV Cube into power and the HDMI cable that goes to the matrix switcher (the big silver box in the rack). Make sure the HDMI is in the same input port number as before. **A brand-new box will show a picture, but the iPad app can't control its apps (open ESPN, etc.) until the manager finishes setting it up — so for a replacement, you'll get the screen working, then text the manager to finish the setup.**

4. On the bartender iPad app, go to the TV control screen for that TV number. Select the correct input source that matches the Fire TV (usually labeled "Fire TV" or the input number).

5. Test by trying to play something on that TV from the app. If the TV now shows the Fire TV home or a stream, it's good.

If none of this worked, take a photo of the TV screen and the Fire TV box (show any lights or labels), and text it to the manager on duty.
